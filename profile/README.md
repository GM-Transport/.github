# Reglamento Interno de Uso de GitHub - GM Transport

Este repositorio contiene las normas y directrices obligatorias para todos los miembros y colaboradores con acceso a la organización de GitHub de **GM Transport**. 

> [!IMPORTANT]
> El incumplimiento de estas normas podrá derivar en la **revocación inmediata** de los accesos.

---

## 1. Identificación y Registro de Cuentas
Para garantizar la trazabilidad y seguridad del código fuente, el perfil de cada usuario debe cumplir con:

* **Correo Electrónico:** Es obligatorio utilizar y verificar una cuenta con el dominio institucional `@gmtransporterp.com`.
* **Nombre de Perfil:** Debe mostrar el **Nombre Completo** o, en su defecto, *Primer Nombre y Primer Apellido*. No se permiten seudónimos.
* **Propiedad de la Cuenta:** Al vincular el correo institucional, el colaborador reconoce que la actividad dentro de la organización está sujeta a la gestión de la empresa. GM Transport se reserva el derecho de administrar estos accesos para fines operativos.

## 2. Uso Exclusivo Profesional
El uso de la cuenta vinculada está limitado estrictamente a actividades laborales.

* **Prohibiciones:** No se permite el uso de la cuenta para proyectos personales, externos o almacenamiento de información ajena a la empresa.
* **Ubicación:** El acceso debe realizarse exclusivamente dentro de las **instalaciones de la empresa**, salvo autorizaciones específicas para trabajo remoto.

## 3. Seguridad y Autenticación
La protección de nuestros repositorios es prioridad máxima.

1.  **Doble Factor (2FA):** Es obligatorio mantener activa la verificación en dos pasos en todo momento.
2.  **Método de Seguridad:** Se debe utilizar exclusivamente una **aplicación de autenticador** (Google Authenticator, Microsoft Authenticator, etc.). 
    * *Nota: Se desaconseja el uso de SMS por motivos de vulnerabilidad.*

## 4. Auditoría y Seguimiento
GM Transport mantiene un esquema de vigilancia activa:

* **Audit Logs:** Se supervisarán constantemente los registros de auditoría para monitorear acciones, accesos y cambios en el código.
* **Validación Administrativa:** El administrador enviará un mensaje personal a cada integrante para confirmar su cuenta definitiva antes de iniciar el seguimiento formal de actividad.

## 5. Compromiso del Usuario
Al ingresar a la organización, el usuario acepta:
* El monitoreo de su actividad.
* Mantener la **integridad y confidencialidad** de la propiedad intelectual de GM Transport.
